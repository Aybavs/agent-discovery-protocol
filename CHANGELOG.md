# Changelog

All notable changes to this project will be documented in this file.

This project is in an early working-draft phase.

## Unreleased

### Added

- Initial repository structure.
- RFC-0000 problem statement — English canonical working draft.
- RFC-0000 problem statement — Turkish translation under `translations/tr/`.
- RFC-0001 technical specification — planned outline placeholder.
- RFC-0002 rationale — planned outline placeholder.
- Manifesto — planned outline placeholder.
- Non-normative schema placeholder under `schema/`.
- Non-normative example placeholders under `examples/`.
- Contribution, code-of-conduct, security, and license documents.

### Notes

- The English RFC-0000 is the canonical version of the problem statement; the Turkish document is a translation of it.
- Files under `schema/` and `examples/` are non-normative placeholders and do not define the protocol; the discovery document format is intentionally deferred to RFC-0001.
- `RFC-0000`, `RFC-0001`, and `RFC-0002` are internal project identifiers, not RFC Editor numbers.
