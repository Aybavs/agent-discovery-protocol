# Discussion 0001: Naming

This project currently uses `ADP` as a codename.

The final name is intentionally unresolved.

## Questions

- Should the project name include the word `agent`?
- Should the discovery resource name include the word `agent`?
- Should the name emphasize `service-interface discovery` instead?
- How can the project avoid conflicts with existing `.well-known` conventions?

## Candidate project names

- Agent Discovery Protocol
- Service Interface Discovery
- Agent Service Discovery
- Machine Interface Discovery
- Web Service Interface Discovery

## Candidate endpoint names

Endpoint naming is deferred to RFC-0001.

Any proposal should consider existing conventions and possible naming conflicts.
