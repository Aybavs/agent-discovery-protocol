# Code of Conduct

This project is intended to be a respectful, technical, and open discussion space.

## Expected behavior

Participants are expected to:

- be respectful and constructive,
- assume good faith,
- criticize ideas rather than people,
- keep discussions focused on the technical subject,
- acknowledge prior work fairly,
- avoid vendor hostility,
- avoid personal attacks,
- avoid harassment, intimidation, or discriminatory language.

## Unacceptable behavior

Unacceptable behavior includes:

- personal insults,
- harassment,
- threats,
- repeated off-topic disruption,
- bad-faith argumentation,
- doxxing or sharing private information,
- discriminatory comments,
- intentionally misrepresenting another participant's position.

## Standards discussion principle

This project may discuss existing standards, protocols, vendors, products, and open-source projects.

Such discussions should remain technical and neutral.

The goal is not to declare winners or losers. The goal is to identify the scope of this project and its relationship to existing work.

## Enforcement

Maintainers may remove comments, close issues, lock discussions, or block participants who violate this code of conduct.

## Reporting

If you see behavior that violates this code of conduct, contact the repository maintainer.
