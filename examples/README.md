# Examples (Non-Normative Placeholders)

These examples are **illustrative placeholders only. They do not define the protocol.**

The discovery document format is intentionally **not specified** until `RFC-0001`. Nothing in these files is normative, and no field name, structure, or location should be inferred from them as part of any specification.

They exist only to reserve a place in the repository structure for future, post-`RFC-0001` examples. Until the discovery model in `RFC-0001` is stable, these files deliberately contain no concrete discovery fields.

See [`../docs/rfc-0000-problem-statement.md`](../docs/rfc-0000-problem-statement.md) for the problem this project addresses.
